String search algorithm visualiser

This app helps with visualising the behaviour of three classic string search algorithms: the simple naive algorithm (which checks the pattern string for a match at all possible positions), the Knuth-Morris-Pratt (KMP) algorithm and the Boyer-Moore algorithm. The latter comes in two flavours: a simple version, known as Boyer-Moore-Horspool that uses only the Bad Character heuristic, and the full version using the Good Suffix heuristic as well.

The code for the full Boyer Moore algorithm has been translated from the C version at wiki.dreamrunner.org.

Since the Boyer-Moore implementations search only ASCII characters, all non-ASCII characters are removed from both the pattern and the strings when using those algorithms.
